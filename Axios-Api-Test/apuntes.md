## Axios API
1. Revisar el sitio oficial de Axios [https://axios-http.com/es/]
2. Instalar axios desde npm con el comando 
```sh
npm install axios
```
3. Ejecutar el proyecto de React en el navegador con el comando
```sh
npm run dev
```
